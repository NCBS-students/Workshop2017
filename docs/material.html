Summary of Day 1 : Sat, 18 November 2017 On the first day, we kicked off the workshop with a brief introduction to programming and its backbone: algorithm development. Once we got a handle of writing an algorithm for finding the maximum in a list of numbers, we used it as an example to cover the basics of programming (variable declaration, conditional statements and for/while loops). After writing our own code for finding the maximum, we went ahead an applied the same approach for finding the first 100 prime numbers.

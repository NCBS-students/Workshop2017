Day 1 : Sat, 18 November 2017 In first day, we will cover basic concepts of programing like variables, constants, data types, conditions, loops and function. After little bit of theory, we will have hands on session with programming langues of your choice. Right now we are planning to have this session with Python , R and MATLAB. We will also have block based visual programming for those who don't want to code in any specific language. You can also suggest if you want to learn programming in specific langues other than mentioned above.

Day 2 : Sat, 25 November 2017 In this session we will concentrate on data-handling, visualization and file I/O. This information will help you to access data from external file, analyze it and visualize it. We will also look at the examples where you can automate repetitive tasks. We will also take a look at how to use third party libraries.
Day 3 : Sat, 2 December 2017 In final day, we will take different kinds of case studies and will show how you can write simple code and achieve your desired output. We will also touch upon briefly on more advance topics like Classes and Objects. If you have any specific example in mind, you can send us details and suggest a case study.

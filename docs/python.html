Day 1

Python is space/indentation sensitive. Be careful while using indentation. We recommend you to use 4 spaces while using any kind of indentation.Every statement should be in separate line (in reality you can divide statement to fit in multiple lines but for now, use thumb of rule as one statement per line). We learned following things in python.
print
Used in printing any kind of data within program.
print("hello there!")
Variables
Used in storing any kind of data. You should NOT use numbers are variable names.
my_int_variable = 1
my_string_variable = "Some Text"
my_boolean_variable = True
my_list = [2,3,4]
For loop
For loop used in iterating over each element in your list.
my_list = [2,3,4,5]
for element in my_list:
   print(element)
Above program will print following output,
2
3
4
5
Range
Range will create list of numbers starting from 0.

for element in range(4):
   print(element)
Above program will print following output,
0
1
2
3
If you want range within specific elements, you can use following syntax,

for element in range(2,6):
   print(element)
It will create elements from 2 to 6 (6 excluded). Above code will give following output.
2
3
4
5
len
Finds length of list (in reality any object with iterating elements.)
my_list = [3,4,5,34]
print(len(my_list))
Above will print "4" on the output screen/ console.
Conditions
You can use conditions to compare two elements
my_list = [2,3,4,5]
for element in my_list:
   if element > 3:
     print(str(element) + " is greater than 3.")
   else if element == 3:
     print("It is 3.")
   else:
     print(str(element) + " is less than 3.")
Above program will print following output,
2 is less than 3.
It is 3.
4 is greater than 3.
5 is greater than 3.

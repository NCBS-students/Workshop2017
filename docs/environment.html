Environmental Setup

For our hands on session, we recommend you to get your own laptop. However if you don't know access to laptop, you can still follow along with one of our volunteers. You can check out all the codes used in this workshop on our GitHub repository which you can always try out later on. This page contains environmental setup for most common operating systems, however if you use any other operating system, please get in touch with us so that we can guide you accordingly. If you have no prior experience, we recommend using Python.

Python

We will use Python 3.6.3. For easier coding experience you can additionally install python IDE like PyCharm Community Edition, Atom, SublimeText 3. For scientific computing, we recommend using jupyter notebook, which allows combining narrative text and equations with live code and data visualization plots (Jupyter notebook).
Windows
If you don't have python installed on your system, you should install Anaconda instead official python binary. It comes with all the standard libraries which you will need in most of the analysis. This also comes with jupyter notebook, and is the easiest way to install python and all other scientific computing libraries.
Ubuntu\Linux
Generally ubuntu\linux will come with pre installed python. You can check which version of python is installed by running following command in terminal
python3 --version
If you don't have Python 3+ installed on your system, you can install it by following command
sudo -E apt-get install python3.6
If you are unable to install it, please follow this tutorial. Additionally, install jupyter notebook using the following commands (in the terminal):
sudo -EH pip3 install --upgrade pip
sudo -EH pip3 install jupyter
To run jupyter notebook, open up a terminal and run:
jupyter notebook
The notebook should open up in your favorite browser.
Mac OS
Mac OS X, Sierra will come with Python 2 pre installed. However for our hands on session you need to install Python 3+. Please follow this detailed tutorial to install python 3. Once python 3 is installed, you can run the same linux commands (above) to install jupyter notebook. Another way of installing python3 in Mac OS might be to use anaconda, similar to windows. Download and and install from here.

R

We will use R version 3.4.2 in our hands on sessions. For easier coding experience it is recommended that you install RStudio.
Windows
Download binary from CRAN website and install it as an administrator. Once r is install, install R studio from the download site.
Ubuntu\Linux
You need to add CRAN minor to your source list. Add following entry in your /etc/apt/sources.list file
deb https://https://ftp.iitm.ac.in/cran/bin/linux/ubuntu artful/
Now you can install R by using following commands.
sudo apt-get update
sudo apt-get install r-base
If you encounter any problem, check this blog post. Once r is install, install R studio from the download site.
Mac OS
Please follow instructions given on official webpage. Once r is install, install R studio from the download site.

MATLAB / GNU Octave

MATLAB is not free software, hence please refer to their official website for downloading instructions. If you are using NCBS/InStem/CCAMP official laptop, you can contact IT section (ithelp@ncbs.res.in) and ask them to install MATLAB for you. You can always try trial version from official website for the duration of the workshop. An open source alternative for MATLAB is GNU Octave, which has similar commands and similar GUI as MATLAB.
Windows
To install GNU octave in windows, simply download and run the exe file from here .
Ubuntu / Linux
To install GNU octave in windows, open up a terminal and run:
sudo -E apt-get install octave
Mac OS
Using package managers like homebrew might be the easiest approach. Once you install homebrew you simply have to open up a terminal and
brew install octave
(not tested. You might have to give super user access). Read more about installing Octave on Mac Os here.
If you face any problem setting up environment, you can always contact us and we will help you.

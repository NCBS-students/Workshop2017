Environmental Setup

For our hands on session, we recommend you to get your own laptop. However if you don't know access to laptop, you can still follow along with one of our volunteers. You can check out all the codes used in this workshop on our GitHub repository which you can always try out later on. This page contains environmental setup for most common operating systems, however if you use any other operating system, please get in touch with us so that we can guide you accordingly. If you have no prior experience, we highly recommend using Python.

Python

We will use Python 3.6.3. For easier coding experience you can additionally install python IDE like PyCharm Community Edition, Atom, SublimeText 3.
Windows
If you don't have python installed on your system, you should install Anaconda instead official python binary. It comes with all the standard libraries which you will need in most of the analysis.
Ubuntu\Linux
Generally ubuntu\linux will come with pre installed python. You can check which version of python is installed by running following command in terminal
python3 --version
If you don't have Python 3+ installed on your system, you can install it by following command
sudo apt-get install python3.6
If you are unable to install it, please follow this tutorial.
Mac OS
Mac OS X, Sierra will come with Python 2 pre installed. However for our hands on session you need to install Python 3+. Please follow this detailed tutorial to install python 3.

R

We will use R version 3.4.2 in our hands on sessions. For easier coding experience you can additionally install RStudio.
Windows
Download binary from CRAN website and install it as an administrator.
Ubuntu\Linux
You need to add CRAN minor to your source list. Add following entry in your /etc/apt/sources.list file
deb https://https://ftp.iitm.ac.in/cran/bin/linux/ubuntu artful/
Now you can install R by using following commands.
sudo apt-get update
sudo apt-get install r-base
If you encounter any problem, check this blog post.
Mac OS
Please follow instructions given on official webpage.

MatLab

MatLab is not free software, hence please refer to their official website for downloading instructions. If you are using NCBS/InStem/CCAMP official laptop, you can contact IT section (ithelp@ncbs.res.in) and ask them to install MatLab for you. You can always try trial version from official website, or you can also install Octave which is open source free alternative of MatLab.
If you face any problem setting up environment, you can always contact us and we will help you.

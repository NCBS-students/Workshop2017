Day 2

After quick recap from previous day, we learned Python functions, file Input/Output and basic data plotting.
Functions
They are very useful in any programming language. These are ideally blocks of code which you can use repetitively. For example, following simple function will print "hello there!".
def say_hello():
  print("hello there!")
In above example, def is reserved keyword to define function in Python. say_hello is the function name. Also notice two brackets and colon in front of function name. That is way of defining function. Brackets are useful in passing some argument in our function. After that, you can write your logic. To use this function in our code, we can use it's name.
say_hello()
Remember to put two brackets in front of function name while calling it in your code.
Now we will pass some argument to our function. Arguments are variables which you can use inside your function.
def say_hello_by_name(name):
  print("hello %s!"%name)
In above example, name is an argument which we will pass on to our function. In Python, you can pass any number and any kinds of arguments into the function.
say_hello_by_name("Rohit") #This will print "hello Rohit!"
say_hello_by_name("Guest") #This will print "hello Guest!"
You can also return some value from the function. Way to do is to use return keyword.
def add_numbers(num1, num2):
  return num1 + num2
In above example, add_numbers function takes two arguments num1 and num2 and returns addition of them.
k = add_numbers(2,3)
print(k) # Prints 5
print(add_numbers(10,10)) # Prints 20
Important thing to remember is that return statement should should be ALWAYS at the end of function. Any sentences written after return keyword will NOT be executed.
File I/O
Many cases you have to work will multiple data files. You need to analyze and visualize data stored inside them. With the help of Python you can very easily achieve this. Here we will specifically look at simple text based files. To handle specialize file formats like "xls", we need some special packages which we will look at next week. Simplest way to read any file inside python can be done in following way,
with open("filename.txt", "r") as file_variable:
  print(file_variable.read())
In any programming language, one has to open file first, then do our task and then close it. In Python, you can do opening and closing automatically with the help of with command. In above example, program will open filename.txt file and print its content. Notice r after name of file. That is called mode of file handling. Here r represents read mode. You can see here to know about available modes and what do they do. You can also open in default mode by not specifying any mode. Once we open our file, we have to assign some variable which we can use as file content. In above example file_variable is that variable. file_variable.read() will read all file content. If you want to read entire file line by line, you can do something like following,
with open("filename.txt", "r") as file_variable:
  for line in file_variable:
   print(line)
Basic Plotting
Data visualization is important aspect of our research. Python provides very flexible ways to plot and visualize your data. For that we will be using python's matplotlib library. If you do not have it , please check here for the instructions. You can use it like following way
import matplotlib.pylab as plt # Import library
x = [2,3,4] # Sample data
y = [10,2,3] # Sample data
plt.scatter(x,y) # This will create scatter plot of x and y
plt.savefig("filename.png", format="png") # Save plot to file
plt.show() # Opens plot
This is very powerful library in python. You can read more about this and few basic examples of different kinds of plots here.
